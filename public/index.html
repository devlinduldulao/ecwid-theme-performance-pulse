Quick Start Guide

Follow these steps to see your real storefront performance data.

1
Enter your storefront URL

Paste the public URL of your Ecwid-powered store (e.g. https://my-store.com).

2
Add audit targets

List the pages you want to measure — homepage, product pages, cart, etc. One per line as "Label | /path".

3
Click "Save Settings" then "Run Audit"

The app calls Google PageSpeed Insights and shows your real scores below — performance, accessibility, SEO, and specific improvement opportunities.

What about "Preview Demo"?

The Preview Demo button shows sample data so you can explore the dashboard layout. It is not real data from your store. A yellow banner will always appear when preview mode is active. Run Audit to get your actual metrics.

No paid services required

Audits run directly from your browser to Google PageSpeed Insights — completely free. Your API key (if provided) never leaves this browser. No backend server or database is involved.
