# Marketplace Assets

This folder contains source artwork for publishing Theme Performance Pulse in an app marketplace.

Included assets:

- `icon.svg`: primary app icon source
- `banner.svg`: hero/banner artwork for listing pages or release announcements
- `screenshot-owner-dashboard.png`: live capture of the owner dashboard UI
- `screenshot-preview-mode.png`: live capture of preview mode with sample audit results

Recommended next step before submission:

- rerun the local preview server and recapture screenshots whenever the UI changes materially
- export raster icon/banner copies in the exact dimensions required by the Ecwid submission form
- keep the vector brand artwork in version control as the editable masters
- use the screenshots in your listing copy and release notes
