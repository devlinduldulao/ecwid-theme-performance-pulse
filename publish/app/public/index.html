Deployment Notes

This package stays static-first. The owner dashboard lives in Ecwid admin, while audits run directly from the browser against PageSpeed Insights.

No hosted backend required

The merchant can save audit targets and run audits from the Ecwid dashboard. Any optional PageSpeed API key stays local to the merchant browser and is not written into Ecwid app storage.
