Theme Performance Pulse

Privacy Policy

Effective date: April 6, 2026

Theme Performance Pulse for Ecwid is a static owner dashboard for Ecwid merchants. The default deployment model does not use a hosted database, Redis cache, or always-on application server.

What the app stores

  • Storefront base URL configured by the merchant
  • Audit target URLs and labels configured by the merchant
  • Preferred audit strategy such as mobile or desktop
  • Preview data and recent audit results stored in the current browser or Ecwid app storage when available

What the app does not collect by default

  • No centralized cross-merchant analytics history
  • No payment information processing
  • No customer database export
  • No private Ecwid token storage in client-side files

Third-party services

When a merchant runs an audit, the dashboard sends a request directly from the merchant browser to the public Google PageSpeed Insights API. Optional API keys remain in the merchant browser and are not written into Ecwid app storage by the default static deployment.

Contact
